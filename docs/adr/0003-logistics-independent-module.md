# ADR 0003 - Logistics as independent module

## Decision

La red local de movilidad sera un modulo independiente, integrado con
operaciones y publicaciones pero no mezclado dentro del marketplace.

## Rationale

La logistica tiene reglas, riesgos, reputacion y validaciones propias.

## Consequences

- Marketplace puede sugerir necesidades de entrega.
- Logistica decide prestadores, propuestas y estados.
- Pagos y comisiones quedan fuera de esta primera etapa.
