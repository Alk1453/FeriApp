# ADR 0001 - Simplified reputation

## Decision

Mostrar reputacion simple y explicable: identidad confirmada, calificacion,
operaciones realizadas, antiguedad y cualidades resumidas.

## Rationale

Una puntuacion opaca puede generar desconfianza. FeriApp necesita que usuarios
con poca experiencia tecnologica entiendan por que confiar.

## Consequences

- Los calculos internos pueden ser ricos.
- La UI debe mostrar solo indicadores claros.
- Cada indicador visible debe poder explicarse.
