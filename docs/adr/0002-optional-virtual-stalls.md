# ADR 0002 - Optional virtual stalls

## Decision

Los puestos virtuales seran opcionales y no seran requisito para crear una
publicacion simple.

## Rationale

Exigir tienda o catalogo agrega friccion a usuarios que solo quieren publicar
algo puntual.

## Consequences

- Una publicacion puede existir sin puesto.
- Un perfil puede tener puesto y tambien publicaciones personales.
- El modelo debe separar publicacion de puesto.
