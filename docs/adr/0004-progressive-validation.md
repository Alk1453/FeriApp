# ADR 0004 - Progressive validation

## Decision

La validacion sera progresiva segun riesgo, categoria, valor, frecuencia y
funciones solicitadas.

## Rationale

Validar demasiado temprano bloquea el uso inicial. Validar demasiado tarde
aumenta riesgo.

## Consequences

- Publicaciones simples de bajo riesgo no requieren documentacion compleja.
- Profesionales, comercios, transporte y categorias reguladas pueden requerir
  evidencia.
- Cada validacion debe registrar evidencia, fecha, responsable y vencimiento.
