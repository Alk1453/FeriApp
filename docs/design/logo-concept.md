# FeriApp - Concepto del logo

## Objetivo

El logo de FeriApp debe transmitir cercania, confianza, economia local y simplicidad.

FeriApp no es un marketplace corporativo ni una plataforma tecnologica fria. Representa:

- la feria del barrio;
- el intercambio entre vecinos;
- pequenos comercios;
- oficios;
- servicios;
- movilidad local;
- comunidad.

## Concepto principal

El simbolo principal es una pequena tienda o puesto de feria.

La estructura de la tienda debe ser el elemento dominante del logo. El nombre `FeriApp`, o una version reducida como `FA`, puede integrarse dentro de la tienda como parte de su fachada.

El usuario debe reconocer la silueta del comercio incluso cuando el texto desaparezca.

## Elementos secundarios

El sistema puede incorporar de forma discreta:

- una planta;
- una caja o paquete;
- una bicicleta;
- senales de intercambio y movilidad local.

Estos elementos no deben competir con el nombre. La tienda siempre debe ser protagonista.

## Estilo visual

Buscar un estilo:

- limpio;
- moderno;
- amigable;
- minimalista;
- legible;
- escalable.

La referencia visual puede tomar claridad de Airbnb, iconografia simple y formas redondeadas, pero FeriApp debe mantener identidad propia ligada al barrio.

## Restricciones

Evitar:

- exceso de detalles;
- degradados complejos;
- sombras fuertes;
- estilos caricaturescos;
- aspecto infantil;
- estetica corporativa rigida;
- elementos tecnologicos genericos.

## Uso esperado

El logo debe funcionar en:

- barra superior;
- favicon;
- icono de aplicacion;
- PWA;
- pantallas moviles;
- modo claro;
- modo oscuro;
- tamanos reducidos;
- marcador de mapa.

## Filosofia

El logo debe transmitir:

> La herramienta digital para quienes nunca tuvieron una.
