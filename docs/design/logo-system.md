# FeriApp - Sistema de logo

## Archivos generados

Los assets oficiales iniciales viven en `public/brand/`:

- `logo-primary.svg`: logo principal con tienda, nombre y lema.
- `logo-horizontal.svg`: uso recomendado para encabezados y barras.
- `logo-vertical.svg`: composiciones verticales o piezas institucionales.
- `logo-icon.svg`: icono de app, avatar y marcador destacado.
- `logo-monochrome.svg`: uso a una tinta sobre fondos claros.
- `logo-dark.svg`: uso claro sobre fondos verdes u oscuros.
- `favicon.svg`: favicon vectorial.
- `favicon-32.png`: favicon pequeno.
- `favicon-192.png`: icono PWA.
- `favicon-512.png`: icono PWA grande.
- `apple-touch-icon.png`: icono para dispositivos Apple.
- `logo-icon.png`: version PNG del icono.

## Concepto

La tienda o puesto de feria es el elemento principal. Debe ser reconocible incluso sin texto.

Los elementos secundarios - hoja, caja y bicicleta - representan crecimiento, intercambio, movilidad y economia local.

## Uso recomendado

- Usar `logo-horizontal.svg` en navegacion, encabezados y presentaciones compactas.
- Usar `logo-primary.svg` cuando haya espacio suficiente.
- Usar `logo-icon.svg` para PWA, favicons, avatar, mapa o marcador.
- Usar `logo-monochrome.svg` cuando se necesite una sola tinta.
- Usar `logo-dark.svg` sobre fondos oscuros.

## Reglas

- No deformar.
- No rotar.
- No cambiar la paleta.
- No eliminar la tienda.
- No agregar efectos 3D, sombras fuertes ni degradados complejos.
- Mantener aire alrededor del logo.
- Priorizar legibilidad en tamanos pequenos.

## Nota de implementacion

La home usa `logo-horizontal.svg` en el panel principal y la metadata usa `favicon.svg`, `favicon-32.png` y `apple-touch-icon.png`.
