# Domain - Sharing and Diffusion

## Principio

Compartir es una capacidad transversal, no una funcion secundaria.

## Elementos compartibles

- publicacion;
- producto;
- servicio;
- perfil;
- puesto virtual;
- seleccion de productos;
- campana;
- promocion;
- necesidad de transporte.

## Canales prioritarios

- WhatsApp;
- Instagram Stories;
- Instagram Reels;
- Facebook;
- enlace publico;
- menu nativo de compartir del dispositivo.

## Recursos generables

- URL publica;
- Open Graph;
- texto breve sugerido;
- tarjeta visual cuadrada;
- imagen vertical para historias;
- codigo QR.

## Restriccion

No prometer publicacion automatica en plataformas que no lo permitan. Cuando no
haya integracion directa, generar el recurso visual y facilitar compartirlo.
